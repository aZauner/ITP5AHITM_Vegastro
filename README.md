# Vegastro 5AHITM


## Starten des Projekts

```
npm i

ionic serve

npm run start:dev

ng serve
```

## Starten von Postgres DB Lokal (Minikube)

```
minikube start

kubectl apply -f .\postgres.yaml

kubectl port-forward deployment/postgres 5432:5432

```
